### Справочная информация
#### Модель Изинга

Модель статистической физики заключающаяся в:
- Геометрической решётке из атомов некой размерности
- Каждый атом выступает как вершина решётки и имеет спин
- Атомы имеют между собой связи

Основные величины:
- Спин
- Энергия решётки (Находится как )
- Температура
- Намагниченность (Находится как сумма всех спинов)
- Теплоёмкость решётки
- Восприимчивость


#### Алгоритм Свендсена–Ванга
Swendsen–Wang — это кластерный (нелокальный) алгоритм Монте‑Карло для моделей Изинга/Поттса, в котором за один шаг строятся кластеры спинов через случайные связи и затем каждый кластер флипается целиком, что резко уменьшает критическое замедление по сравнению с локальными спин‑флип алгоритмами типа Метрополиса или Гаусса–Кобо.

Шаг алгоритма:
1. Установление связей
- Просматривают все пары ближайших соседей (i,j).
- Если $S_i \neq S_j$ связь не ставится
- Если $S_i = S_j$ между ними проводят связь $b_{ij}=1$ с вероятностью $p = 1 - exp(-\beta J)$ или $1 - exp (-2\beta) при J = 1$ иначе $b_{ij}$
- $\beta = \frac{1}{k_BT}$ или $\beta = \frac{1}{T}$
2. Формирование кластеров
- По связям $b_{ij} = 1$ строят кластеры
3. Обращение кластеров
- Для каждого кластера независимо выбирают новое значение спина
- Все спины внутри кластера принимают это значение
